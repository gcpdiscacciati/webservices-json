# webservices-json
## Instalação e uso
- Para o projeto WebserviceCEP, em um ambiente configurado (Ex.: [IDE Eclipse](https://www.eclipse.org/downloads/)), execute o arquivo ConsultaCEP.java, no diretório WebserviceCEP/src/gcpd/webservices/cep/. O programa executará e dará as instruções em caixas de diálogo. Este projeto foi concebido usando a JDK 16.

- Para o projeto Consulta-API-GitHub, em um ambiente configurado com o Python 3, certifique-se que o módulo `requests` está instalado. Caso seja necessário, instale-o com:
#### `pip install requests`
- Em seguida, navegue até o diretório do projeto e basta executar o programa com:
#### `python3 seguidores_github.py`
